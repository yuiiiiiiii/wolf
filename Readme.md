# LyingMan
一款在线狼人杀游戏平台  
仅支持八人游戏，房间号必须为4位数，仅供简单演示使用
### 运行
- 安装[NodeJS](https://nodejs.org/en/)
- 在LyingMan目录下执行 `node server`
- 浏览器浏览 `localhost:81`
- 支持electron `electron web`
